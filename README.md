## Temática

## Descripción
En este sitio web se puede consultar sobre diversos artistas musicales y su informacio personal (nacimiento,origen,nombre,fallecimiento). Tambien se puede consultar distintos albumes lanzados a lo largo de su carrera, incuyendo informacion del nombre del mismo album,lanzamiento,genero musical,durancion y cantidad de canciones que incluye.
